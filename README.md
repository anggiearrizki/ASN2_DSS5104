# DSS5104 Machine Learning and Predictive Modelling  
## Assignment 2: Fraud Detection Project

This repository implements and compares classical and deep learning anomaly detection methods on two datasets: **IEEE-CIS Fraud Detection** (primary) and **PaySim** (secondary).

## Repository Structure
 
```
ASN2_DSS5104/
├── 1_fe.ipynb              # IEEE-CIS: feature engineering & temporal splitting
├── 1b_paysim_fe.ipynb      # PaySim:   feature engineering
├── 2_ml.ipynb              # IEEE-CIS: classical ML models (LightGBM, XGBoost, IsoForest, LOF)
├── 2b_paysim_ml.ipynb      # PaySim:   classical ML models
├── 3_dl.ipynb              # IEEE-CIS: deep learning models (AE semi/supervised, MLP) — run on Kaggle
├── 3b_paysim_dl.ipynb      # PaySim:   deep learning models
├── 4_result.ipynb          # Combined results, cost analysis, error analysis
├── data_raw/               # Raw data — NOT committed (download manually, see below)
│   ├── IEEE-CIS/ieee-fraud-detection/
│   │   ├── train_transaction.csv
│   │   └── train_identity.csv
│   └── PaySim/
│       └── PS_20174392719_1491204439457_log.csv
├── data/                   # Processed parquet files — generated by notebooks
├── results/                # Model output CSVs — generated by notebooks
├── report/                 # Final charts and tables — generated by notebooks
├── requirements.txt
└── README.md
```

## Environment Setup
 
### 1. Create and activate a virtual environment (recommended)
 
```bash
python -m venv .venv
# Windows:
.venv\Scripts\activate
# macOS/Linux:
source .venv/bin/activate
```
 
### 2. Install dependencies
 
```bash
pip install -r requirements.txt
```
 
### 3. Create required output directories
 
```bash
mkdir -p data results report
```
